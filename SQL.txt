SQL Def.
Structured Query Language eignet sich dazu, Datenbanken nach Tabelleninhalten zu fragen.

Datenbanken
Klassische Datenbanken (im Gegensatz zu non-SQL Datenbanken) gibt es im Amateurbereich vor allem in den Ausprägungen mySQL 
(umsonst) und Microsoft SQL (geprochen MS Sequel (Server) - kompfortabler, teuer). Die Installation eines Server (Programm)s ist nötig, 
um Datenbanken verfügbar zu machen. Vorher erscheinen Datenbanken nicht verwendebare Dateien. Bindet man diese Dateien in eine Serverinstanz 
ein - dazu braucht man die Berechtigung das zu tun: i.e. die Rolle admin - kann man der Datenbank Tabellen hinzufügen.
Eine Tabelle hat Spalten eines Typs und Einträge, die den Spalten entsprechen.
Am besten ändert man Tabellen und Spalten (hinzufügen und löschen) in einer GUI. MSSQL Management Studio oder mySQL PhPAdmin stehen dafür zur 
Verfügung. Rechtsklick add/delete.
Nach dem Inhalten fragt man über SELECT * FROM Tabelle WHERE Tabelle.Spalte = wert. Hinzufügen geht mittels INSERT INTO TABLE VALUES
